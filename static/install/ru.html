Установка на iOS / iPadOS

Шаг 1: Установите приложение

Откройте App Store и установите V2RayTun (рекомендуем) или Streisand

Шаг 2: Скопируйте VLESS-ссылку

Вернитесь в бот и скопируйте VLESS-ссылку. Она выглядит так:

vless://xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx@...

Шаг 3: Импортируйте конфигурацию

Откройте приложение V2RayTun → нажмите "+" (вверху справа) → выберите "Import from clipboard"

Шаг 4: Подключитесь

Выберите добавленный сервер из списка и нажмите Подключиться. При первом подключении iOS запросит разрешение на добавление VPN-конфигурации — нажмите "Разрешить"

Шаг 5: Готово!

В статусной строке появится иконка VPN. Теперь вы можете свободно пользоваться интернетом!

Приложения для всех платформ

iOS: V2RayTun Streisand V2Box
Android: v2rayNG
macOS: V2RayTun Streisand Nekoray
Windows: Nekoray V2RayN
Linux: Nekoray

Процесс аналогичен на всех платформах: установите приложение → импортируйте VLESS-ссылку → подключитесь

Важные замечания

  • Один конфиг работает на всех ваших устройствах одновременно
  • Сохраните VLESS-ссылку в надежном месте
  • Проблемы с подключением? Попробуйте перезапустить приложение
  • После подключения проверьте работу, открыв любой сайт
